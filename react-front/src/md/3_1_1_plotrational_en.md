# Rational Function

A rational function is of the form

$y = \frac{P(x)}{Q(x)}$

where $P(x)$ and $Q(x)$ are polynomial functions.

Here, the final form of the function will be:

$y = \frac{a_{P}x^2 + b_{P}x + c_{P}}{a_{Q}x^2 + b_{Q}x + c_{Q}}$
